❓ Pertanyaan Umum (FAQ)

📌 Bagaimana cara melaporkan fasilitas rusak? ▼
Isi form di sebelah kiri: pilih jenis fasilitas, lokasi, deskripsi, foto (opsional), dan ambil koordinat GPS. Lalu klik "KIRIM LAPORAN". Laporan akan masuk ke daftar.
🛠️ Bagaimana status laporan diperbarui? ▼
Tim sarpras (atau demo) bisa mengubah status dari "MENUNGGU" menjadi "DIPROSES" lalu "SELESAI". Klik tombol "Ubah Status" di tiket laporan untuk simulasi progres.
📸 Apakah wajib menyertakan foto? ▼
Tidak wajib, tetapi sangat direkomendasikan agar tim teknis lebih cepat memahami kerusakan. Foto disimpan secara lokal (base64) di penyimpanan browser.
🌐 Apakah lokasi GPS diperlukan? ▼
Ya, untuk membantu petugas menemukan lokasi tepat kerusakan. Tekan tombol "Ambil Lokasi Saya" dan izinkan akses lokasi pada perangkat.
🔍 Bagaimana cara mencari laporan lama? ▼
Gunakan kotak pencarian di atas (ikon 🔍) — ketik kata kunci seperti nama gedung, jenis kerusakan, atau status. Daftar akan langsung tersaring.
💾 Apakah data laporan tersimpan secara permanen? ▼
Data disimpan di localStorage browser Anda. Untuk aplikasi riil, laporan akan dikirim ke server backend. Saat ini hanya demo offline.
